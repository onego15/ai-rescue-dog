# AIレスキュードッグ

## 概要

自律型AIエージェント（犬型ロボット）が3D迷路環境で救助対象を見つけて救助するゲームです。

## 特徴

- **3D迷路環境**: Three.jsを使用した3D迷路
- **自律型AIエージェント**: センサーで周囲を認識し、A*アルゴリズムで最適経路を計画
- **経路探索**: A*アルゴリズムによる自律的な経路計画とリプランニング
- **リアルタイムUI**: ステータス表示とミニマップによる状況確認
- **外部アルゴリズム**: エージェントの行動アルゴリズムをMarkdownファイルで定義・管理

## 技術スタック

- Three.js: 3Dグラフィックス
- Vite: 開発サーバーとビルドツール
- Vanilla JavaScript: ゲームロジック

## インストール

```bash
npm install
```

## 実行方法

開発サーバーを起動:

```bash
npm run dev
```

ブラウザで http://localhost:5173 を開きます。

## ビルド

```bash
npm run build
```

## AIエージェントの動作

1. **環境認識**: エージェントはセンサー範囲内（8マス）の情報のみを認識
2. **経路計画**: 救助対象を検出すると、A*アルゴリズムで最適経路を計算
3. **リプランニング**: 移動中に新しい情報を得ると経路を再計算
4. **自律移動**: 計画した経路に沿って自律的に移動

## ゲームの目標

AIエージェントが迷路内の救助対象（赤い人型NPC）を見つけて救助することです。エージェントは完全自律で動作し、プレイヤーの操作は不要です。

## アルゴリズムのカスタマイズ

エージェントの行動アルゴリズムは `algorithms/agent-behavior.md` ファイルで定義されています。

### アルゴリズムファイルの構造

Markdownファイル内に以下のJavaScript関数をコードブロックとして定義できます：

- `detectTarget(agentPos, targetPos, sensorRange)`: ターゲット検出ロジック
- `decide(agent, targetPos)`: 意思決定ロジック
- `move(agent, deltaTime)`: 移動制御
- `shouldReplan(agent, targetPos)`: リプランニング判定

### アルゴリズムの変更方法

1. `algorithms/agent-behavior.md` を編集
2. コードブロック内の関数を変更
3. ブラウザをリロード

アルゴリズムの読み込みに失敗した場合は、自動的にデフォルトの実装にフォールバックします。
